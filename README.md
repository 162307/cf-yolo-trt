# cf-yolo-trt  
理论支持yolo全系列模型  
基于yolo-trt的穿越火线ai自瞄  
相关功能都已封装成类，方便后续开发使用  
目前有三个类  
dxgi_cap类，负责截图  
Yolo类，负责推理  
pid_control类，负责鼠标移动  
可使用model文件夹下的trt文件  
如需使用自定义trt文件  
pt转trt可参考https://github.com/Linaom1214/TensorRT-For-YOLO-Series  
cmake编译可参考https://github.com/Monday-Leo/Yolov5_Tensorrt_Win10
